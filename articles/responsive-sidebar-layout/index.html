Since the first websites were published, there was a layout with sidebar. I really like sidebars. Sidebars are great for navigation or just containing some (short) information.

For my blog, I want to create a sidebar, which looks nice on every screen. In addition, I want just a blog without hundreds of features. That’s the reason why I choose a static site generator, SASS and Pug (earlier Jade). I won’t use any pre-processor now, because for these few line of code, it’s useless.

I placed comments between the line of codes, to describe what the following part of code makes. For expressing length, I use em or rem. These are relative units, more here. Next, I use flexible boxes, which are a CSS3 feature, all about that.

I have linked the result here and following is the source:
